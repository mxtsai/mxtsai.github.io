# Pre-Processor Documentation

  * Need to install [PyDicom](https://pydicom.github.io/pydicom/stable/index.html)
  
  Files Format
  ~~~
  PreProcessing\
    |--DICOM_Files\
      |--(patient ID, ex. 4015008183865)\
        |-- (a lot of DICOM files for that patient)
      ...
    |-- Program\
      |-- pacs_to_python.py
      |-- main.py
      |-- augument_and_upload.py
  ~~~
