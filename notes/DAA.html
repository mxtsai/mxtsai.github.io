Review of Graph Theory:
Graph G is dened by a pair of sets (V,E)
V- is a set of vertices (|V| = n)
E- is a set of edges (|E| = m) where each edge can have a weight
General Denitions:
1) A and B (vertices) are adjacent if they are connected by an edge
2) deg(A) is the amount of neighbors of A
3) A path is a sequence of vertices v1,v2, such that each subsequent vertices are adjacent
4) A path is simple if all its vertices are distinct
5) A length of a path is the number of edges in that path
6) Distance between A&B is the shortest path between them (no path )
7) A cycle = closed path = start and end at same vertex
8) A cycle is simple if all vertices (not counting start and end) are distinct
9) A graph is connected if any two vertices can be joined by a path (else disconnected)
10) Each disconnected graph is called a connected component
Spanning Tree - subset of using all the vertices of with the minimum number of edges
Properties: (1) fully connected without cycles
(2) a connected, undirected graph has at most spanning trees
(3) a disconnected graph cannot have a spanning tree!
